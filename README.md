# Excel Splitter 📊

**Excel Splitter** — это десктопное приложение на Java с графическим интерфейсом (GUI), предназначенное для разделения объемных файлов Excel с большим количеством списка строк на более мелкие файлы с меньшим количеством строк.  Программа автоматически сохраняет строку заголовка в каждом новом файле и поддерживает экспорт в различные форматы. 
Программа создана преимущественно для разделения Excel файла выгрузки списка товаров и его количества из 1С на Ламода, которая в свою очередь принимает Excel файлы современного формата, но с определенным количеством строк (500) в одном файле.

## ✨ Ключевые возможности

* **Настройка разделения:** Пользователь сам задает количество строк для каждого нового файла.
* **Сохранение заголовков:** Первая строка исходного файла (заголовок) автоматически переносится во все созданные файлы.
* **Мультиформатный экспорт:** Поддержка сохранения результатов в форматах:
    * `.xlsx` (Современный Excel)
    * `.xls` (Excel 97-2003)
    * `.csv` (Текст с разделителями-запятыми)
    * `.txt` (Текст с разделителями-табуляцией)
    * `.xml` (Структурированные веб-данные)
* **Информативный UI:** Встроенный индикатор прогресса (Progress Bar) и окно логов в реальном времени для отслеживания процесса обработки больших файлов.
* **Фоновая обработка:** Тяжелые операции выполняются в отдельном потоке, что предотвращает зависание интерфейса.

## 🏗 Архитектура и Паттерны проектирования

Код написан с соблюдением разделения ответственности (Separation of Concerns). В проекте применены следующие паттерны ООП:

* **MVC (Model-View-Controller):** Графический интерфейс (`ViewMainFrameFS`) полностью изолирован от бизнес-логики (`ServiseES`). Взаимодействие между ними осуществляется через `AppController`.
* **Strategy (Стратегия):** Логика сохранения вынесена в отдельные классы (например, `ExcelExporterStrategy`, `CsvExpoterStrategy`), объединенные общим интерфейсом. Это избавляет код от громоздких `switch`-конструкций и позволяет легко добавлять новые форматы (Принцип Open/Closed).
* **Factory (Фабрика):** `SaveExporterStrategyFactory` инкапсулирует логику создания нужной стратегии в зависимости от выбранного пользователем формата.

## 🎯 Использование
1. **Запустите программу и нажмите Browse... в первой строке, чтобы выбрать исходный Excel файл.**

2. **Укажите желаемое количество строк на один файл (например, 500).**

3. **Введите базовое имя для новых файлов или оставьте поле пустым(к файлам автоматически добавятся индексы _1, _2 и т.д.).**

4. **Нажмите Browse... во второй строке, чтобы выбрать папку для сохранения результатов.**

5. **Выберите нужный формат файла в выпадающем списке.**

6. **Нажмите Split и следите за прогрессом в окне логов!**

7. **После завершения папка с сохранёнными файлами откроется автоматически.**
